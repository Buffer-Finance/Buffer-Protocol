## Prerequisities

-   [Node.js v12.20.2][1]

## Install

```bash
npm install
```

## Tasks

### Start ganache

```bash
npm run ganache
```

### Run tests

```bash
npm run test
```

### Generate code coverage

```bash
npm run coverage
```
